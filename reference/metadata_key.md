# Build RespiLens metadata key from `model_out_tbl` data

Build RespiLens metadata key from `model_out_tbl` data

## Usage

``` r
metadata_key(model_out_tbl)
```

## Arguments

- model_out_tbl:

  Forecast tibble from
  [`get_fcast()`](https://accidda.github.io/acciddasuite/reference/get_fcast.md).

## Value

Named list for RespiLens metadata key
